Explain what the terminal is in your own words.
The terminal is a commands interface that lets you interact directly with your computer using typed commands instead of a graphical interface.

What is the major difference between Git and GitHub?
Git is a version control tool that tracks changes locally. GitHub is an online platform where you can store and share Git repositories.

What are the steps to fork a GitHub repository?
Go to the repository on GitHub, click fork in the top right corner, choose where to save it, then clone the fork to your computer.
